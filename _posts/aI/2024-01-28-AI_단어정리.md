## ㄱ

<div id="결정_계수"></div>
- 결정 계수 : R^2 = 1 - (타깃 - 예측 )^2의 합 / (타깃 - 평균)^2의 합

<div id="과대적합"></div>
- 과대적합 : 훈련 세트에서는 좋은 성능이 나오나, 테스트 세트에는 나쁜 상황 (훈련 세트에 최적화)

<div id="과소적합"></div>
- 과소적합 : 테스트 세트가 훈련 세트보다 성능이 더 좋거나, 둘다 부진한 경우 (학습 부적)

<div id="결측치"></div>
- 결측치 : 데이터 값이 없는 것

<div id="기댓값"></div>
- 기댓값 : 확률 과정을 무한히 반복했을 때 얻을 수 있는 평균 독립변수

<div id="가우스-마르코프_정리"></div>
- 가우스-마르코프 정리 : 오차항이 [**독립 동일 분포**](#독립_동일_분포)를 따르고 [**기댓값**](#기댓값)이 0이고, 독립변수가 [**외생변수**(#외생변수)]일때 [**최소 제곱 추정량**](#최소_제곱_추정량)은 [**최선의 선형 불편 추정량**](#최선의_선형_불편_추정량)이다.



## ㄴ

<div id="노름"></div>
- 노름 : Norm. 벡터의 크기 또는 측정하는 방법을 뜻함

## ㄷ

<div id="데이터_클러스터링"></div>
- 데이터 클러스터링 : 비슷한 특징을 가진 데이터를 그룹으로 묶는 것

<div id="독립_동일_분포"></div>
- 독립 동일 분포 : [**i.i.d**](#iid)확률 변수들이 서로 독립적이고, 각각의 동일한 확률분포를 따른다

## ㄹ

<div id="로젠블랫"></div>
- 로젠블랫 : 프랭크 로잰블랫. 미국의 신경생물학자

## ㅁ

<div id="머신러닝"></div>
- 머신러닝 : [**인공지능**](#인공지능)의 한 분야. 컴퓨터가 학습 데이터를 통해 스스로 학습하고, 그 결과를 바탕으로 예측, 결정하는 기술

<div id="맨해튼_거리"></div>
- 맨해튼 거리 : 각 차원에서의 절대적인 변화량을 합산한 값

<div id="마할라노비스_거리"></div>
- 마할라노비스 거리 : Mahalanobis Distance. 데이터 분포를 고려하여 거리를 측정하는 방법

<div id="모델_파라미터"></div>
- 모델 파라미터: 알고리즘이 찾은 값

## ㅂ

<div id="비지도_학습"></div>
- 비지도 학습: 입력값만을 이용해 데이터의 패턴이나 구조를 찾아내는 학습방법

<div id="비모수"></div>
- 비모수 : 데이터가 정규분포를 따르지 않거나, 모를 때 사용하는 통계적 분석법

<div id="비용_함수"></div>
- 비용 함수 : 손실을 수치화하는 함수

## ㅅ

<div id="샘플링_편향"></div>
- 샘플링 편향 : 학습 데이터가 전체 데이터를 대표하지 못한 경우


## ㅇ

<div id="인공지능"></div>
- 인공지능 : 사람처럼 생각하고 학습하고 추론할 수 있는 지능이 있는 프로그램/시스템

<div id="인스턴스_기반_학습"></div>
- 인스턴스 기반 학습 : 훈련 데이터를 기억하고, 이를 바탕으로 예측을 수행하는 학습 방법

<div id="유클라디안_거리"></div>
- 유클리디안 거리 : 두 점 사이의 직선거리. 일반적으로 알고 있는 거리 공식을 통해 구한 값

<div id="외생변수"></div>
- 외생변수 : 모델 외부에서 결정되는 종속 변수에 영향을 주는 변수

## ㅈ

<div id="전문가시스템"></div>
- 전문가시스템 : 전문가의 지식과 추론 능력을 컴퓨터 프로그램으로 구현한 [인공지능](#인공지능) 시스템

<div id="지도학습"></div>
- 지도학습 : 입력값과 정답값을 함께 학습하여 새로운 입력값에 대한 출력값을 예측하는 학습 방법
